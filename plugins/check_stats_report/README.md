# check_stats_report

This script is used to check website stats on awstats host.

|                 |                                 |
| --------------- | ------------------------------- |
| Language        | Bash                            |
| Source          | Atomia custom                   |
| Original author | jimmybergman@atomia.com         |

---



## Requirements
None

## Usage

### Parameters
```
check_stats_report.sh
    <some_linux_site>
    <min_hourly_visits>
    <hours_process_delay>
```

### Exit codes
```
    0 - OK
    2 - CRITICAL
    3 - UNKNOWN
```
